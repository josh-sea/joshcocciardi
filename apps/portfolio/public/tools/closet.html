Recommended System: ClosetMaid Impressions

Why this system: Available at Home Depot/Lowe's, modular, works with drywall installation, consistent look across all three walls.

Shopping List:

  • ClosetMaid Impressions 25" Tower (x2) - for drawer units on Arts & Crafts wall (~$200 each)
  • ClosetMaid Impressions 16" Shelving Kit - for Games wall (includes brackets, shelves) (~$120)
  • ClosetMaid Impressions 16" Shelving Kit - for Costume wall upper/lower (~$120)
  • ClosetMaid Impressions Hanging Rod - 72" for costumes (~$25)
  • ClosetMaid 3-Drawer Fabric Bins (x3-4) - for costume accessories (~$25 each)
  • Sterilite Clear Storage Bins - various sizes for all walls (~$100-150 total)
  • Simple Houseware Stackable Baskets - for lower costume wall (~$30-40)
  • Label Maker - Brother P-Touch (~$30)

Installation Notes:

  • ClosetMaid uses a vertical hanging rail system that mounts to wall studs
  • You'll need to locate studs and mount the rails every 16-24" depending on load
  • Everything else clips into the rails, making it reconfigurable
  • Use drywall anchors for areas between studs if needed
  • Total estimated cost: $1,000-1,200 for complete system

Pro Tips:

  • Install LED strip lighting under the top shelves on each wall for great visibility
  • Keep a step stool in the room (the Gorilla 2-step is perfect and folds flat)
  • Label everything with clear labels at kid eye level
  • Put most-used items at kid height (24-48")
  • Use clear bins wherever possible so you can see contents
  • Leave the middle floor area clear for a small table/workspace if desired later
