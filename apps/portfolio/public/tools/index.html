Welcome to Your Tools Directory

This is a password-protected area where you can deploy and access your development tools and utilities.

How to Add New Tools:

1. Create HTML files in the public/tools/ directory

2. Include <script src="auth.js"></script> in your HTML

3. Add <script>protectPage();</script> to protect the page

4. Run npm run build to build the project

5. Deploy with firebase deploy
