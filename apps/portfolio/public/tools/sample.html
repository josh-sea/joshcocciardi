Welcome to the Sample Tool Page

This is a demonstration of how to create protected HTML pages in the tools directory. This page is automatically protected by the authentication system.

How This Works

Every page in the /tools directory can be protected by including these two scripts:

<script src="auth.js"></script>
<script>protectPage();</script>

When a user tries to access this page:

  • The auth.js script checks if they're authenticated
  • If not authenticated, they're redirected to the login page
  • After logging in with admin/admin, they gain access
  • Authentication persists for the browser session

Creating Your Own Tool Pages

To create a new tool page:

  1. Create a new HTML file in public/tools/ directory
  2. Include the authentication scripts as shown above
  3. Add your custom functionality and styling
  4. Run npm run build to build the project
  5. Deploy with firebase deploy
